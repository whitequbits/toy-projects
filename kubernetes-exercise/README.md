# Kubernetes Exercise

## Getting Started

This is my project to learn kubernetes work

### Prerequisites

What things you need to use it

```bash

1. Node JS 14
2. Docker
3. Kubernetes

```

## Local Access

- Kubernetes Service (localhost:80)
- Docker Compose (localhost:8080)

## How to Run it in Local

```bash
# using docker-compose
docker-compose up -d --build

# using kubectl
kubectl apply -f deployment.yaml
kubectl apply -f service.yaml
```
