# world_time

A simple project to learn flutter

## Getting Started

This project is my starting point to learn Flutter.

I learn it from this [Youtube Playlist](https://www.youtube.com/playlist?list=PL4cUxeGkcC9jLYyp2Aoh6hcWuxFDX6PBJ)

### Prerequisites

What things you need to use it

```bash
1. flutter
2. Android Studio
3. Xcode
```

## How to Run it in Local

1. Setup your Flutter in local
2. Install Android Studio and Xcode
3. Install Plugin in Vscode
4. Run debug
5. Start Emulator
